ファイルの名前を一括変更.xls　Ver.1.1
　作：坂江　保　　　　　　　2014/09/12

　Excelなので簡単・安心の操作性。連番名作成・置換・変換機能が簡単操作で行えます。

///主な機能///

　　指定した任意のフォルダ内のファイルの名前を一括で変更します。以下は主な機能です。

　　■連番作成

　　　　任意の文字列＋連番の作成が行えます。連番では「0001」といった設定も可能です。

　　■文字列フィルター

　　　　任意の文字列を元に対象とするファイルを取捨選択できます。
　　　　任意の文字列の含まれるファイルを、対象として残すか、消去かといった設定が可能です。

　　■置換・変換

　　　　指定したファイル名の置換（一部置換を含む）や、大文字から小文字、ひらがなから
　　　　カタカナといった変換が可能です。



///その他機能///

　　・変更後のファイル名の重複をチェック（自動）
　　・変更後のファイル名と指定フォルダのファイルとの重複チェック（自動）
　　・ファイルに使用できない文字列チェック（自動）
　　・並べかえ（昇順・降順・文字数順）
　　・フォルダ内のファイル数表示

　　など



///基本的な使用法///

　　　１．ブックを立ち上げマクロを有効にする

　　　２．フォームが表示されるので「フォルダ選択」で該当フォルダを選択する

　　　３．選択フォルダ内のファイルが表示されるので、隣のセルに新しい名前を設定する

　　　４．フォーム上の「ファイル名の変更」ボタンを押す。完了



///留意事項///

　　　・ソフト基本画面上の「！」ボタンを押すと、使用上の留意点が表示されます。

　　　・1度に変更できるファイル数は3000です。

　　　・このソフトはセーブしないで終了して下さい。
