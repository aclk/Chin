DB設定書き換えツール
作者：仁茂田　正明
■目的
　DBの設定を書き換えるのがめんどくさいので作った

■準備
boot.batとDBSetting.jarとdbSetting.propertiesを一緒のフォルダにぶち込めばOK

■起動方法
Jarファイルをダブルクリックして実行できる環境ではJarファイルを、
でなければバッチファイルをダブルクリックすれば起動します。

起動した画面で、
　JdbcUrlを選択しない（空っぽ）だったらJDBCのURLは書き換えません。
　UserIdだけ入力したらPassword、schemaはUserIdと同じのが設定されます。

■コマンドラインでの使用
　使い方は java -jar DBSetting.jar help を見てください。

■プロパティファイルの指定
　dbSetting.propertiesと違う名前のプロパティファイルを指定したい場合はコマンドラインで指定できます
　（指定しない場合はデフォルトでdbSetting.propertiesが使用される）

例：depsetting.propertiesを読み込ませたい場合
java -Ddb.config=depsetting -jar DBSetting.jar
＃指定する場合は[.properties]の部分はいりません。

■Eclipseとの同期（GUIのみ）
Eclipseで開いているプロジェクトと同期するためにプロパティファイルに以下の分を追加すればおけ！

#----------------- ここから -----------------
#Eclipseで閉じているプロジェクトらしきものはリスト表示しない
#（GUIモードでprj.rootでEclipseのワークスペースを指定した場合のみ有効）
sync.eclipse=true
#----------------- ここまで -----------------



■履歴機能
GUIモードでプロパティファイルに以下の分があれば履歴を指定した件数分だけ保存します。
#----------------- ここから -----------------
#履歴の保存件数（0なら履歴を取らない）
history.size=5
#----------------- ここまで -----------------
履歴から選択する場合は、OKボタン横にHistoryボタンがあるので押下すると履歴から選択ダイアログがでます。
履歴から選択ダイアログで使用する内容を選べばメイン画面に値が反映されます。
＃モーダルダイアログなので、メイン画面のOKボタンを押下したい場合は閉じること
履歴を消したい場合はdb.historyファイルを削除すればOK
