リンクテーブル作成ツール-海賊版.mdb　使用方法 			2007年7月24日 三浦一仁

■前提条件

「クライアント環境構築ガイド」における
「VII	Oracle Instant Client」の章の設定が完了している事を前提とします。


■使い方

1,リンクテーブル作成ツール-海賊版.mdb を、
自身のローカルマシン上にコピーします。

2,Accessにてリンクテーブル作成ツール-海賊版.mdbを開くと、
起動時に「接続情報」フォームが表示されます。
画面上のデータソース名、テーブル名、パスワードを自身の個人スキーマの情報を入力したのち、
「フルコース」ボタンを押下してください。

3,「〜時間がかかります〜」という警告メッセージが表示されます。「はい」ボタンを押下してください。

4,「テーブル名をカンマ区切りで指定してください」というインプットボックスが表示されますので、
リンクテーブルを作成したいテーブル名(Oracle上の物理名)をカンマ区切りで指定し、OKボタンを押してください。

※結合データ作成作業のうち、このリンクテーブル作成ツールを使用する時には、
テンプレExcel作成ツール、結合テストFWかどちらか用に、applicationContext-join-test.xml
の設定ファイルを作成しているはずです。ファイル中の
	<property name="tableIds" value="〜〜
から始まるカンマ区切りのテーブル列挙を、そのままコピー＆ペーストしてください。

5,リンク作成作業を行います。
テーブル数により変動しますが15分以上はかかります。
ほったらかしてお茶など飲んでひとまず忘れてください。

6,「リンクテーブルの作成が終了しました。」というメッセージが表示されると、
正常にリンク作成が終了しています。後は通常のMDBと同じ使い方をしてください。

上記以外のメッセージが表示された場合、エラーですので、
メッセージに従った対応の後、再度「フルコース」ボタンのクリックにより、実行して下さい。


以上。
