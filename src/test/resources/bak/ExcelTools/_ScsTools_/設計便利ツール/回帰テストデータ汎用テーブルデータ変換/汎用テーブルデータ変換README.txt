回帰テストデータの汎用テーブルデータをまとめて変換してしまうツールです。

テストで汎用テーブルデータを挿入する時に、数値項目に数値が入っていなかったり、日付の形式が違っていると、マテリアライズド・ビューに変換される際にエラーが発生します。それを避けるために、回帰テストデータの該当するデータのみをエラーが出ないように変換してしまいます。


使用方法

1) [汎用テーブルデータ変換]ボタンをクリック。
2) 対象となる回帰データフォルダを選択。
3) フォルダの中の全Excelファイルについて、[事前データ]シートと[期待DB]シートに含まれる、CR0100のデータが変換される。


ver 1.0 2008/12/16
・CR0102の[カラムタイプ区分]が"N"で始まるものを数値、"D"で始まるものを日付として扱い、対応するCR0100のデータを変換する。
・CR0100の数値項目が数値以外の場合は 0 を挿入。
・CR0100の日付項目に "YYYY/MM/DD" 形式の値が入っていれば、"YYYYMMDD"形式に変換。
・CR0100の日付項目に"[10]YYMM"形式の値が入っていれば、"20YYMM01"または"19YYMM01"に変換。

ver 1.1 2008/12/16
・汎用テーブル"CDF"のＳＥＱＮＯ． 100,200,300のレコードの内容がすべて、"U10","OTHERZ"," "の場合に200,300の内容をそれぞれ"U20","OTHERZ"," "、"U30","OTHERZ"," "に変換。(一意制約違反を避ける)

ver 1.2 2008/12/18
・汎用テーブルデータ定義([CR0102])を回帰データから読むのではなく、SCHM_M001の定義データから読み込み。その定義を元に変換する方式に変更。

ver 1.3 2009/01/29
・ver 1.1の機能および、その他の追加機能の有効/無効を[実行機能選択]画面で切り替え可能にした。
