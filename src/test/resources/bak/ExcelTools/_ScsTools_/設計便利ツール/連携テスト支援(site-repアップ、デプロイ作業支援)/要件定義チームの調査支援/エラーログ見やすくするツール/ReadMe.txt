regrep 使用方法

2008年6月6日 K.Miura

■これはなに？

テキストファイルを指定すると、
あらかじめ定義した辞書テキストファイル(excludeList.txt)の文字を含む行を除外し、
結果をテキストファイルへと保存してくれるツールです。

■前提条件

Java5以降がインストールされているコンピュータ。
javaコマンドがどこからでもたたける設定がしてあること。

■主な使い方

コマンドラインから、
  regrep.bat [対象のファイルパス]
とコマンドを実行してください。

または、regrep.bat に対して、
対象のファイルをドラックアンドドロップしてください。

regrepフォルダに、result.txt という結果ファイルが出力されます。


■辞書の編集

同フォルダにある excludeList.txt を編集してください。
