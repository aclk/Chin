checkdiffmodule 使用方法

2008年6月9日 K.Miura

■これはなに？

Spiritsデプロイ作業時の差分チェック結果テキストファイルの二つを、
引数に指定すると差異部分を抽出し出力するツールです。


■前提条件

Java5以降がインストールされているコンピュータ。
javaコマンドがどこからでもたたける設定がしてあること。

■主な使い方

コマンドラインから、このツールのフォルダに移動し、
  checkdiffmodule.bat [対象ファイルその1のパス] [対象ファイルその2のパス]
とコマンドを実行してください。

または、checkdiffmodule.bat に対して、
対象のファイル２つを選択、ドラックアンドドロップしてください。

このフォルダに、result.txt という結果ファイルが出力されます。
