regrep 使用方法

2008年6月9日 K.Miura

■これはなに？

Spiritsデプロイ作業時の依存性チェック結果テキストファイル(out.txt)を、
引数に指定すると重複部分を抽出し出力するツールです。


■前提条件

Java5以降がインストールされているコンピュータ。
javaコマンドがどこからでもたたける設定がしてあること。

■主な使い方

コマンドラインから、このツールのフォルダに移動し、
  checkduplicationmodule.bat [対象のファイルパス]
とコマンドを実行してください。

または、checkduplicationmodule.bat に対して、
対象のファイルをドラックアンドドロップしてください。

このフォルダに、result.txt という結果ファイルが出力されます。
