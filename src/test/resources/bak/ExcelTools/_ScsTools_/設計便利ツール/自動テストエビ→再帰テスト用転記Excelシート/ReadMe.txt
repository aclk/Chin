自動テストエビ→再帰テスト用転記.xls　使用方法 			2007年11月1日 三浦一仁

■前提

「プロダクトテストガイド.doc」における
「VIII.回帰テスト」「1.回帰テスト準備」「(2)処理後DB」の作業を
自動化したツールです。


■使い方

1,本Excelファイル「自動テストエビ→再帰テスト用転記.xls」を、
自身コンピュータの任意の場所にコピーしてください。


2,Excelにてファイルを開き、「設定シート」を編集してください。

各フィールドは、以下の情報を入力してください。

フォルダ：
自動テスト後、「<プロダクトID>PGMTest-execute-Evidence<パターンID>.xls」の
エビデンスが出力されるフォルダのフルパス。

ファイル名のパターン：
対照としたいエビデンスExcelファイル名のパターンIDより、左の部分
(<プロダクトID>PGMTest-execute-Evidence)を指定してください。

対象外にする列パターン：
転記しないDB列名を指定してください。

3,「設定シート」から「実行」ボタンをクリックしてください。

4,「事後DB(データ転記結果)」に、再帰テスト用データシートの書式でデータが転記されます。
結果をコピー＆ペーストして再帰テスト用の「事後DB」シートに張ってください。
